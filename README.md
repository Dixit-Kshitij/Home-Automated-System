# Home Automation System using Arduino

A smart home automation system built using Arduino that allows both
manual (Bluetooth-based) and automatic (sensor-based) control of home
appliances like lights, fan, and gate.

## FEATURES - Dual Mode Operation: Manual (Bluetooth) and Automatic
(Sensor-based) - Smart Fan Control using temperature (DHT11) - Automatic
Lighting using LDR - Smart Gate using ultrasonic sensor - Bluetooth
control via mobile - Real-time LCD display

## COMPONENTS - Arduino Uno - HC-05 Bluetooth Module - DHT11 Sensor - LDR -
Ultrasonic Sensor (HC-SR04) - Servo Motor - DC Motor - LCD with I2C

## BLUETOOTH COMMANDS O - Open Gate C - Close Gate L - LED ON l - LED OFF
F - Fan ON f - Fan OFF S - Send Status

## WORKING Manual Mode: - Control via Bluetooth commands

## Automatic Mode: - Temperature controls fan - Light controls LED -
Distance controls gate

RESULTS - Controlled multiple appliances - Real-time automation -
Reduced manual effort
